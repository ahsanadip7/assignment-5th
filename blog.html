What is the delegation in the contex of the DOM, and why is it useful?

Delegation in the context of the DOM (Document Object Model) refers to a technique where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of attaching individual event listeners to each child element, the parent element can "delegate" handling the events of its children.
